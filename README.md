# Library Management System

A full-stack library management system built with Django REST Framework (backend) and React + TailwindCSS (frontend). 
This system allows students and librarians to manage books, borrowings, returns, fines, and user profiles efficiently.

## Features

- User registration and login
- Add, edit, delete, and view books
- Manage book categories and authors
- Borrow and return books
- Track overdue fines
- Role-based dashboards for students and librarians

## Technologies Used

Backend: Django, Django REST Framework, SimpleJWT, PostgreSQL / SQLite  
Frontend: React, TailwindCSS, Vite, Axios
